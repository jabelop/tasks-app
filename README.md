# tasks-app
Backend for a tasks application

## Description

Tasks application backend for managing users, roles, and tasks.

## SaaS Features
Two subscription plans have been impelemented:
 - Free: rate limited to 6 per minute and a max of 100 tasks asociated.
 - Premium: No rate limited and a max of 100000 tasks asociated.


## Installation

```bash
$ npm install
```

## Before running the app

```bash
# Copy and configure .env file
cp .env.example .env 
```
Put your own values.
Then you can get up your full environment with the Docker Compose step

### Development with Docker Compose
You can use the docker-compose file to get all the environment up and then run the migrations & fixtures step.
The docker compose command will get up all the needed containers, mailpit, redis, postgre and api.
Then you will have to run the migrations and fixtures step. 

```bash
# get up the full environment with docker compose
docker compose up 
```

### Run migrations & fixtures
Once you get all your containers up (with docker compose or manually) you must run these commands on a separate terminal.
```bash
# run the db migrations
npm run db:migration:run
# run the db fixtures
npm run db:fixtures
# User data to login after fixtures
# user: admin
# password: secret

#user: user
#password: secret
```
